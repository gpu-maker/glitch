# REALITY GLITCH OS

A browser-based simulated government terminal containing a digital anomaly.

## Features

- Terminal-style OS interface
- Persistent digital entity
- Entity mood + learning behavior
- Fake apps and filesystem
- Fake internet browser
- Text-only psychological experience
- LocalStorage persistence
- Modular architecture

## Run Locally

1. Download repository
2. Open `index.html` in a browser
3. No server required

## Controls

- Type commands in terminal
- `help` — show commands
- `apps` — list apps
- `open <app>` — launch app

## Storage

Uses LocalStorage to persist:

- entity memory
- chat history
- notes
- system state
